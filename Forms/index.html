HTML Forms

Form is an HTML element with which we can receive input from the user, we can use in a form various interactive elements such as: "Text input", "numbers", "values", "email" and "password" where the user can enter information. In addition to the alphanumeric input, we also have "checkboxes", "radio buttons" and "buttons" that we can add to a form.

Action attribute
We can also add an action attribute to the form, which we use to indicate the destination of the content. This can be an absolute or relative URL.
For Example <form action="test.php" method="post">

Method attribute
In addition to the action attribute, there is also the method. This will determine what HTTP request will be used
GET: Used to request data from a specified source.
POST: Used to send data to a server.
For Example <form action="test.php" method="post">

The HTML <form> element can contain one or more of the following form elements:

HTML Form Elements

Tag Description
<input>-----> With this tag we can create alphanumeric input fields, checkboxes and radio buttons.
<textarea>--> Defines a multiline input control (text area). This tag creates a text field where the user can enter long text.
<label>-----> This shows text associated with the <input> element such as password, username, etc.
<fieldset>--> This tag will draw a box around a group of elements.
<legend>---> This has a subscript for elements in a <fieldset>
<select>----> Defines a drop-down list
<optgroup>-> Defines a group of related options in a drop-down list. We will use this tag to define a group of options.
<option>---> This tag is used to add an option to a dropdown list such as <select> or <datalist>.
<button>---> This tag creates a button to perform other functionality such as submitting the form.
<datalist>--> The datalist is similar to the select. You can select an item from a list, but as a user you also have the option to write text yourself
<output>---> The output tag will return a value from an operation (the result of a calculation)
